Online Lecture Exemption Application System

  • This is the best website i have ever made because there are lots of new functionalities has been introduced in this website compared to the two previous system.
  • Students can access this system to apply for lecture exemption.
  • The purpose of the exemption is to allow students to engage in other important activities during lecture hours.
  • Common reasons for exemption include convocation events, club or association activities, and MUET exams.
  • The system aims to digitalize the manual application process, which typically takes around 3+ days for approval from higher authorities in UiTM Johor.
  • Over 2000 students in UiTM Johor utilize this system.
  • The system is developed with a responsive design, allowing it to be accessed on both mobile and web-based platforms.
  • This system has has authentication functionalities which will give access to anyone who have registered and signed-in into their account
  • This system is connected to a database through phpMyadmin and Utilized data protection through Sha1 Hash codes for simple encryption
  • Consisted of interface for customers and also for admin that allow them to manage the system easily(Dashboards)
  • The system is developed with a responsive design, allowing it to be accessed on both mobile and web-based platforms.
  • The system is utilize email notification funtionalities through PHPMailer library
  • Has printing and downloading pdf functionalities
  • Display data in a comprehensive table structure and table are filterable based on search filters
  • Passwords can be changed if desired
